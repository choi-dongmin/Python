# 파이선 (패키지 매니저)

패키지 매니저 란? 패키지란 소프트웨어 즉 프로그램이다. 

우리는 프로그래밍을 하면서 혼자서 모든것을 할 수는 없다 그러므로 우리는 다른 개발자가 만든 소프트웨어를 하나의 부품처럼 사용한다 그 부품을 우리는 패키지라고 하고 또한 Library 라고 한다.

이 패키지 매니저란 다른 사람들이 만든 부품을 쉽게 다운받고 내가 만드는 프로그램에 부품으로 적용 시킬 수 있게 도와주는 프로그램을 '패키지 매니저' 라고 일컫는다.

그럼 이 패키지 매니저에는 어떠한 것들이 있을까?

언어들 마다 각각 패키지 매니저가 있지만 파이선은 PIP 라고 하는 패키지 매니저가 있다.

그리고 우리가 찾는 패키지를 찾고 확인 할 수 있는곳이 pypi.python.org 이곳에서 마치 App Store 처럼 내가 찾는 패키지를 찾아 볼 수 있다.

## 키워드

패키지 매니저 : 패키지 매니저란 프로그래밍을 하면서 다른 사람들이 만든 객체 또는 프로그램을 자신의 프로그램에 도움을 주는 프로그램이다.
